ورودی ماتریس:

adj_matrix = np.array(...): ماتریس مجاورت را تعریف می‌کنیم. برای سادگی، در این کد ماتریس به صورت دستی وارد شده است. برای گرفتن ماتریس از ورودی کاربر می‌توانید از دستوراتی مانند input() استفاده کنید.

وارد کردن کتابخانه‌ها:

import networkx as nx: این خط کتابخانه networkx را برای کار با گراف‌ها وارد می‌کند و به آن یک نام کوتاه (nx) می‌دهد.

import matplotlib.pyplot as plt: این کتابخانه برای رسم نمودار و شکل است.

import numpy as np: این کتابخانه برای کار با آرایه‌ها و ماتریس‌ها بسیار سریع و کارآمد است.

ساخت گراف:

G = nx.from_numpy_array(adj_matrix): این دستور مهم‌ترین بخش کد است! این خط ماتریس مجاورت ما را به یک شیء گراف تبدیل می‌کند که networkx آن را می‌فهمد و می‌تواند روی آن عملیات انجام دهد.

رسم گراف:

pos = nx.spring_layout(G): این خط به networkx می‌گوید که چطور نقاط گراف را روی صفحه مرتب کند تا زیباتر و خواناتر باشد.

nx.draw_networkx_nodes(...)، nx.draw_networkx_edges(...) و nx.draw_networkx_labels(...): این سه دستور، به ترتیب نقاط، یال‌ها (خطوط) و شماره نقاط را روی صفحه می‌کشند. ما با استفاده از پارامترهایی مثل node_color، node_size و font_size ظاهر آن‌ها را مشخص می‌کنیم.

plt.title(...) و plt.axis('off'): عنوان نمودار را تعیین کرده و محورهای مختصات را مخفی می‌کنند تا نمودار تمیزتر دیده شود.

plt.show(): در نهایت، این دستور پنجره‌ای را باز می‌کند که گراف رسم شده در آن نمایش داده می‌شود.

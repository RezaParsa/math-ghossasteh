تعریف ماتریس اصلی:

adj_matrix: جدول (ماتریس) اصلی ماست که می‌خواهیم متمم آن را حساب کنیم.

پیدا کردن ابعاد:

n = len(adj_matrix): تعداد سطرها را پیدا می‌کند. چون ماتریس مجاورت مربعی است، این عدد همان تعداد ستون‌ها هم هست.

ساخت ماتریس متمم:

complement_matrix = []: یک لیست خالی می‌سازیم تا نتیجه را در آن قرار دهیم.

حلقه‌های اصلی:

for x in range(n) و for y in range(n): دو حلقه تودرتو می‌سازیم تا به تمام خانه‌های ماتریس اصلی دسترسی داشته باشیم. x شماره سطر و y شماره ستون است.

بررسی خانه‌ها:

if x == y:: این شرط بررسی می‌کند که آیا خانه فعلی روی قطر اصلی قرار دارد یا نه. اگر بود، مقدار آن را تغییر نمی‌دهیم و همان مقدار اصلی را در ماتریس متمم قرار می‌دهیم.

else:: اگر خانه روی قطر اصلی نبود، مقدار آن را برعکس می‌کنیم.

if adj_matrix[x][y] == 1:: اگر مقدار 1 بود، 0 را به ماتریس متمم اضافه می‌کنیم.

else:: اگر 0 بود، 1 را اضافه می‌کنیم.
